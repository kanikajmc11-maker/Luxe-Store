**LuxeStore – Premium Fashion & Lifestyle E-Commerce**

LuxeStore is a modern e-commerce frontend project built with React and Vite. The goal of this project was to design a clean, responsive, and production-ready storefront UI that demonstrates strong frontend development practices.

**Features**

Multi-Page Shopping Experience
The application includes six complete pages that simulate a real e-commerce website:
•	Home – Hero carousel, featured products, categories, and deals
•	Shop – Full product catalogue with filters and sorting options
•	Product Detail – Product gallery, pricing, selection options and reviews
•	Wishlist – A page for saved products
•	About – Brand story, statistics and company values
•	Contact – Contact form, FAQ section and business information

**Premium UI / UX Design**
A strong focus was placed on visual polish and usability.
•	Elegant Cormorant Garamond display typography
•	Smooth hover animations and transitions
•	Clean product cards and layout structure
•	Mobile-first responsive design
•	Premium Pink + White + Blue color palette

**Shopping Features**
The project simulates a real shopping flow:
•	Slide-over cart drawer with quantity controls
•	Wishlist functionality using a heart toggle on product cards
•	Toast notifications to give instant user feedback
•	Savings summary inside the cart

**Smart Product Discovery**
Users can easily find products through multiple tools:
•	Category filtering
•	Tag-based filtering
•	Price range filtering
•	Search query
•	Sorting by:
o	Trending
o	Newest
o	Price
o	Rating
o	Popularity

**Interactive Elements**
To make the interface feel dynamic and engaging:
•	Hero carousel with auto-play and keyboard navigation
•	Deal of the Day section with a live countdown timer
•	Interactive hover states across the interface

**Getting Started**
Before running the project, make sure you have:
•	Node.js v18 or later
•	npm v9 or later

**Installation**
Clone the project and install dependencies:
npm install
Run the development server:
npm run dev
Build the project for production:
npm run build
Preview the production build locally:
npm run preview


**SEO Optimisation**
The project is built with basic SEO best practices in mind.
Features include:
•	Unique page titles and descriptions via React Helmet
•	Sitemap support
•	Semantic HTML structure
•	Accessible interactive elements with ARIA labels
