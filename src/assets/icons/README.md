# Icons Folder

Place all custom SVG icons and logo assets here.

## Current icons used
All interface icons are from `lucide-react` (already installed).
Import them like: `import { ShoppingBag, Heart } from 'lucide-react'`

## Custom icons to add here
- `logo.svg` — Brand logo SVG
- `payment-visa.svg`, `payment-upi.svg` — Payment method icons
- `brand-*.svg` — Any brand-specific logos

## Usage
```jsx
import { ReactComponent as Logo } from '../assets/icons/logo.svg'
// or
import logoUrl from '../assets/icons/logo.svg'
```
